1991 — As Tags

    História da HTML

    O que é HTML?

    HyperText Markup Language é a linguagem de marcação usada para criar páginas da web. Ela permite estruturar conteúdo com títulos, textos, imagens, links, listas, formulários e muito mais.

    1991 — A criação

    O HTML foi criado por Tim Berners-Lee, um cientista britânico do CERN
    (Centro Europeu de Pesquisas Nucleares)
    Ele desenvolveu a primeira versão do HTML como parte de um sistema chamado World Wide Web (WWW), para compartilhar documentos entre cientistas. Essa versão era muito simples: contendo 18 tags.

    1995 — HTML 2.0

    A primeira versão padronizada foi o HTML 2.0.
    publicada pela IETF (Internet Engineering Task Force).
    Adicionou suporte a formulários, listas ordenadas, e outros elementos básicos.

    1997 a 1999 — HTML 3.2 e 4.01

    A responsabilidade passou para o W3C (World Wide Web Consortium).
    O HTML 3.2 adicionou suporte a tabelas, scripts (JavaScript), e folhas de estilo (CSS). O HTML 4.01 trouxe melhorias na acessibilidade, separação entre estrutura e estilo, e foco em padrões web.

    Anos 2000 — Quase esquecido

    O HTML parou de evoluir por um tempo porque a ideia era substituí-lo pelo XHTML, uma versão mais rígida e baseada em XML.
    Mas o XHTML era complicado e muita gente não adotou.

    2008 em diante — HTML5

    O HTML5 foi lançado como uma resposta moderna às novas necessidades da web.
    Ele trouxe novas tags semânticas como: <header>, <footer>, <section>, <article >.
    Suporte a áudio, vídeo, gráficos com <canvas>, geolocalização, armazenamento local e muito mais. Melhor integração com CSS3 e JavaScript.
